World Wide Web Consortium (W3C)

Created in 1994, the World Wide Web Consortium (W3C) is an international community where Member organizations, full-time staff, and the public work together to develop Web standards. The W3C develop HTML and CSS standards that allow for websites to look and work the same in different web browsers. The W3C is currently led by Tim Berners-Lee, a computer scientist who is best known as the inventor of the World Wide Web.

HTML (Hypertext Markup Language) and CSS (Cascading Style Sheets) are two of the core technologies for building Web pages. HTML is responsible for the structure of the web page, whereas CSS is responsible for the visual layout.



HyperText Markup Language (HTML)

  • Publish online documents with headings, text, tables, lists, photos, and much more.
  • Retrieve online information via hypertext links.
  • Transaction forms for services, searching information, ordering products, etc.
  • Publish online spread-sheets, videos, sounds, and much more.


Cascading Style Sheets (CSS)

  • Colors, layout, and fonts.
  • Adaptation to different types of devices with different screen sizes.
  • Separate from HTML.
  • Makes it easier to maintain websites.
  • Share style sheets across pages and tailor websites to different environments.
